Ye Concert Preservation Project

A public archive project for collecting, organizing, and preserving Ye concert footage so performances do not disappear into dead links, private drives, and takedowns.

What This Is

The goal is to consolidate recordings from Ye concerts into one durable, navigable archive: full videos, setlists, timestamps, source credits, and notes on what has been preserved.

This page is a placeholder while the archive model is designed. The first public build starts with SoFi Stadium Night 2.

Funding Promise

I am committed to documenting this process transparently. I am not taking profit from this project. Any donation money goes toward hosting, storage, bandwidth, and related server costs for preserving the footage.

As the archive grows, this page will list monthly infrastructure costs, money received, and what each payment covered.

Current Cost Model

The current setup uses Oracle Cloud Object Storage for the video file. Storage is relatively cheap at this scale; the main thing to monitor is outbound video traffic as more concerts and viewers are added.

  • Current SoFi fileAbout 18 GB stored on Oracle Object Storage
  • Storage costRoughly under $1/month for this one file
  • Video deliveryOracle lists the first 10 TB/month outbound from North America as free
  • Archive growthCosts scale with total stored footage and traffic
